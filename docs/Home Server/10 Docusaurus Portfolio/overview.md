# Vitrina Profesional

## Dependencias
- **GitHub Actions**: Build en push a `main`.
- **Nginx**: Servidor web con Brotli compression.
- **Plausible Analytics**: Métricas sin cookies.
- **Algolia**: Búsqueda instantánea.

## Tecnologías Clave
- **Docusaurus v3**: React + MDX
- **Theming**: Modo oscuro/light personalizado.
- **SEO**: Sitemap automatizado, meta tags.
- **i18n**: Soporte para español/inglés.
